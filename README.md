# Le Défi de Clem

Mini-jeu HTML/CSS/JavaScript prêt pour GitHub Pages.

## Ce qui a été ajouté

- Sons de jeu : départ, récupération d'un enfant, crash, victoire.
- Le score augmente uniquement quand Clem récupère un enfant.
- Les étoiles ont été remplacées par des enfants dessinés en canvas.
- À la victoire, le jeu souhaite un bon anniversaire à Clem et lance la musique personnalisée si elle est présente.

## Ajouter ta musique d'anniversaire

1. Mets ton fichier MP3 dans le dossier `assets`.
2. Renomme-le exactement : `anniversaire-clem.mp3`.
3. Publie le dossier sur GitHub Pages.

Le chemin doit être :

```text
assets/anniversaire-clem.mp3
```
